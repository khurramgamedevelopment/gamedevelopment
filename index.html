Project Overview

FPS is an action-based 3D game where the player takes the role of the main character to destroy AI enemies. The gameplay is action-based with no strategic or role-playing elements; instead, the game entirely provides a rush of adrenaline. The only goal is to kill all enemies until even one is not left before they kill our character. The version in this project consists of story mode and infinite attack mode. Enemies are trained using Artificial intelligence rather than classical approaches. FPS is developed using Unity, a game development engine.
